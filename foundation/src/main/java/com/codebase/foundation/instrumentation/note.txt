1. 切换目录
    cd foundation/target/classes
2. 添加META-INF/MANIFEST.MF文件,文件内容
    Manifest-Version: 1.0
    Premain-Class: com.codebase.foundation.instrumentation.Agent
3. 打包
    jar cvf instrumentation.jar com/codebase/foundation/instrumentation/Main.class com/codebase/foundation/instrumentation/Service.class
    mv instrumentation.jar ..
    jar cvfm instrumentation_agent.jar META-INF/MANIFEST.MF com/codebase/foundation/instrumentation/Agent.class com/codebase/foundation/instrumentation/Transformer.class
    mv instrumentation_agent.jar ..
4. 运行
    cd ..
    java -cp instrumentation.jar com.codebase.foundation.instrumentation.Main
    java -javaagent:instrumentation_agent.jar -cp instrumentation.jar com.codebase.foundation.instrumentation.Main
